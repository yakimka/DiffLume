## Getting Started

Get it via pip:

```bash
python3 -m pip install --user DiffLume
```

To kick things off, just pop this in your terminal:

```bash
difflume
# or
python3 -m difflume
```

More of a Docker person? We got you:

```bash
docker run -it --rm yakim/difflume
```
