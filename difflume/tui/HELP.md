# DiffLume

DiffLume is a tool for visualizing the differences between two files.

## Keybindings

| Key    | Action                                  |
|--------|-----------------------------------------|
| ?      | This screen                             |
| f      | Make current panel full screen (toggle) |
| c      | Center text in panels (toggle)          |
| F1     | Open new file in left panel             |
| F3     | Open new file in right panel            |
| Ctrl^C | Quit                                    |
